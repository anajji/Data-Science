# Code Book

Explanation of the different data fields in the tidyData.txt

## ID
* activity : the activity of the subject during the measurement process including :
  * WALKING
  * WALKING_UPSTAIRS
  * WALKING_DOWNSTAIRS
  * SITTING
  * STANDING
  * LAYING

* subject : it identifies the subject who performed the activity for each window sample. Its range is from 1 to 30.
